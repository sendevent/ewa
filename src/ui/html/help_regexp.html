Here you need to create RegExp. About sysntax and using you can read in Qt's documentation page.
With regexp you can describe what part of web page you need.

Example Page html source:
<html>
<head>
<title>Exmaple page title</title>
</head>
<body>
 <b>Target</b> content
</body>
</html>
To match page's title, you can use regexp like this:
"Pattern": <title>(.*)</title>
"Case sensetive" checkbox - unchecked
"Minimal" checkbox - checked.
"Replace with" - some text, wich'll replace matched value.
If you'll leave "Replace with" empty, after processing result page wil be without title;
If you'll input in "Replace with" text "NEWTITLE" ( without quotes ), after processing result page's title will be "NEWTITLE";
If you'll input in "Replace with" text "% is changed" ( without quotes ), after processing result page's title will be "Exmaple page title is changed";
